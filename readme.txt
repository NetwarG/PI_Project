После выполнения команды vagrant up, будет создана ВМ с ubuntu 20.04, а такде будет выполнен ряд базовых настроек для полноценной работы приложения на javascript. В число настроек входят:
1) Установка MongoDB, включающая в себя подъем локальной базы с заполнением ее данными из облачного хранилища, а также создание сервиса обновляющего по таймеру (каждый день в 6 утра) облачное хранилище данными из локальной базы.
2) Подъем локального smtp сервиса для отправки писем с ссылкой для активации новых аккаунтов. В качестве библиотеки для работы почтового сервиса был выбран postfix из-за простоты настройки.
3) Установка node js, распаковка архива с проектом и запуск самого проекта, включающий установки всех необходимых javascript библиотек и сборку frontend + backend схемы в 1 целое
4) Установка диспетчера процессов PM2, предназначенного для приложений Node.js. PM2 позволяет преобразовывать приложения в демонов, чтобы они работали как службы в фоновом режиме + запускались при запуске системы.

Для проверки работоспособности проекта, можно перейти по адресу http://localhost:8080/
